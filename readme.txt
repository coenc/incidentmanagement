INCIDENT MANAGEMENT TOOL

Incident management tool is een web-based applicatie gerealiseerd met het Yii framework (www.yiiframework.com). Het Yii framework is een high-performance PHP framework dat de mogelijkheid biedt tot het realiseren van een MVC architectuur op basis van een relationele database. De applicatie 'Incident management tool' biedt gebruikers de mogelijkheid om via een webinterface incidenten te melden aan een projectteam. Vervolgens worden de binnengekomen incidenten toegewezen aan leden van het projectteam onder vermelding van een prioriteit. 
Iedere verandering van status, update, toewijzing worden in de database gelogd t.b.v. de management informatie. Managementinformatie is in deze release niet opgenomen.

Login met gebruikersnaam en wachtwoord demo/demo voor de normale weergave en met admin/admin voor extra beheer faciliteiten, zoals het aanmaken van nieuwe projecten, statussen en prioriteiten.

Voor gebruik op een Apache webserver dient het Yii framework en MySQL  geinstalleerd te zijn. Het script voor het aanmaken van de database met voorbeelddata is opgenomen in het bestand incidentmanagement.sql.
